# Bibliothéque Numérique MIT Projet Professionnel

> Bibliothéque Numérique est une application Java.

## Fonctionnalités

#### Le bibliothécaire permet de : 

* lister les auteurs,
* lister les emprunts, 
* lister les achats 
* lister les livres
* ajouter des livres
* fixe les delais d'inscription pour les utilisateurs simples
* fixe des amendes de retard d'emprunts
